openapilint release notes

0.3.0
-----
* Add implementation of `no-path-item-parameters`.

0.2.1
-----
* Fix bug where a path `parameters` object was being treated like an operation.

0.1.1
-----
* Fix some bad docs.
* Add some missing entries to `package.json`.

0.1.0
